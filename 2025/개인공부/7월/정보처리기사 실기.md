값에 의한 호출(call by value) 개념
**혼동 포인트**
- 함수 내부의 `t`는 `swap` 전용 임시 변수
- `swap`이 원본 변수의 주소를 받지 않으므로(포인터 미사용)    
- `main`의 `a`, `b` 값은 변경되지 않음
```c
void swap(int *a, int *b) {
    int t = *a;
    *a = *b;
    *b = t;
}
// 호출: swap(&a, &b);
```
