문제 1: 별로 삼각형 그리기

문제 설명:

  • 숫자 N이 주어지면, 별로 삼각형을 그리는 프로그램을 작성하세요.
  • N의 범위 0≤n≤100\leq n\leq 100≤n≤10﻿
  • N은 삼각형의 높이를 나타냅니다.
  • 입력은 input.txt 파일에서 읽어오고, 출력은 output.txt 파일에 저장해야 합니다.

풀이 팁:

  • 이중 for 루프를 사용하여 첫 번째 루프는 줄 수를, 두 번째 루프는 별을 출력합니다.
  • 첫 번째 루프는 1부터 N까지 반복하며, 두 번째 루프는 각 줄마다 출력할 별의 개수를 제어합니다.
  • printf()를 사용해 별을 출력하고, 각 줄의 끝에 \n을 추가해 줄바꿈을 합니다.

입력:

  • input.txt 파일의 첫 번째 줄에 정수 N이 주어집니다.

입력 예시 1 (input.txt):

1

출력 예시 1 (output.txt):

*

입력 예시 2 (input.txt):

3

출력 예시 2 (output.txt):

*
**
***

입력 예시 3 (input.txt):

5

출력 예시 3 (output.txt):

*
**
***
****
*****

 * *

문제 6: X 모양 별 그리기

  • 숫자 N이 주어지면, X 모양의 별을 그리는 프로그램을 작성하세요.
  • N의 범위 3≤n≤103\leq n\leq 103≤n≤10﻿ 이면서, 홀수
  • N은 X 모양의 높이를 나타냅니다.
  • 각 행별로 가장 마지막으로 출력된 * 뒤로는 공백을 출력하지 않습니다.

  • 각 줄마다 printf()로 줄바꿈을 추가하여 출력 형식을 맞춥니다.

입력:

  • input.txt 파일의 첫 번째 줄에 정수 N이 주어집니다.

입력 예시 1 (input.txt):

3

출력 예시 1 (output.txt):

* *
 *
* *

입력 예시 2 (input.txt):

5

출력 예시 2 (output.txt):

*   *
 * *
  *

*   *

7
