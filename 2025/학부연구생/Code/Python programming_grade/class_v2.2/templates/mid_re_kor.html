8주차: 중간고사-재시험

🧪 2025학년도 중간고사 - 파이썬 프로그래밍 (재시험)

시험 시간: 60분

문항 수: 5문항

총점: 100점 (문제당 20점)

제출 형식: 각 문제별 .py 파일 작성

입출력 방식: input.txt에서 입력, output.txt에 출력

🔸 문제 1. 대소문자 변환기

🧾 문제 설명

영어 문자열이 입력되면, 각 문자의 대소문자를 반대로 바꾼 문자열을 출력하세요.

📥 입력 조건 및 범위

  • 문자열 길이는 1 이상 100 이하
  • 입력은 영어 대소문자로만 구성됩니다.

📁 입출력 형식

Testcaseinput.txt 내용output.txt 내용
testcase1HellohELLO
testcase2PYthonpyTHON

🔸 문제 2. 짝수 개수 세기

🧾 문제 설명

쉼표로 구분된 정수들 중 짝수의 개수를 출력하세요.

📥 입력 조건 및 범위

  • 정수는 0 이상 100 이하
  • 정수는 1개 이상 100개 이하
  • 입력은 쉼표(,)로 구분됩니다.

📁 입출력 형식

Testcaseinput.txt 내용output.txt 내용
testcase11,2,3,4,52
testcase210,20,303

🔸 문제 3. 별 계단 출력기

🧾 문제 설명

정수 N을 입력받아, 아래와 같이 N줄에 걸쳐 별(*)을 계단 형태로 출력하세요.

📥 입력 조건 및 범위

  • 정수 N은 1 이상 20 이하

📁 입출력 형식

Testcaseinput.txt 내용output.txt 내용
testcase13*
**
***
testcase25*
**
***
****
*****

※ 줄 바꿈 주의, 마지막에 빈 줄 없음

🔸 문제 4. 알파벳인지 확인하기

🧾 문제 설명

입력된 한 문자가 알파벳(영어 대문자 또는 소문자)이면 "Alphabet"을 출력하고, 그렇지 않으면 "Not Alphabet"을 출력하세요.

📥 입력 조건 및 범위

  • 입력은 정확히 1개의 문자입니다.
  • 입력은 알파벳, 숫자, 특수문자 중 하나입니다.

📁 입출력 형식

Testcaseinput.txt 내용output.txt 내용
testcase1AAlphabet
testcase2gAlphabet
testcase35Not Alphabet
testcase4!Not Alphabet

🔸 문제 5. 짝수 홀수 판별기

🧾 문제 설명

정수 N이 주어졌을 때, 짝수면 "Even", 홀수면 "Odd"를 출력하세요.

📥 입력 조건 및 범위

  • 정수 N은 1 이상 1000 이하

📁 입출력 형식

Testcaseinput.txt 내용output.txt 내용
testcase110Even
testcase27Odd
