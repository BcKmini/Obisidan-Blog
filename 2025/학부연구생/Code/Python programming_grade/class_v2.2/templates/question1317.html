문제 13: 문자열 뒤집기와 대소문자 변경

문제 설명:

  • 주어진 문자열을 뒤집고, 각 문자의 대소문자를 반대로 바꿔서 출력하는 프로그램을 작성하세요.
  • 문자열의 길이는 최대 100글자 입니다.
  • 문자열은 input.txt 파일에서 읽어오며, 출력은 output.txt 파일에 저장해야 합니다.

팁:

  • 문자열을 뒤집기 위해서는 문자열의 마지막 문자부터 하나씩 처리하면 됩니다.
  • 대소문자 변환은 각 문자의 아스키 코드값을 활용하거나, C/C++에서는 tolower()와 toupper() 함수를 사용할 수 있습니다.
  • Python에서는 upper()와 lower()함수를 사용해보세요.

입력:

  • input.txt 파일의 첫 번째 줄에 문자열이 주어집니다.
입력 (input.txt)출력 (output.txt)
HelloWorldDLROwOLLEh
C++ProgrammingGNIMMARGORp++c
AbCdEfGhIjKkJiHgFeDcBa
DataScienceECNEICsATAd
PalindromeEMORDNILAp

문제 14: 배열에서 짝수와 홀수 구분하기

문제 15: 문자열에서 모음과 자음 세기

문제 설명:

  • 주어진 문자열에서 모음과 자음의 개수를 각각 세어서 출력하는 프로그램을 작성하세요.
  • 문자열의 길이는 100글자 입니다.
  • 문자열은 input.txt 파일에서 읽어오며, 출력은 output.txt 파일에 저장해야 합니다.

팁:

  1. 모음과 자음의 구분:
    • 모음은 영어에서 a, e, i, o, u 다섯 가지로 정의됩니다.
    • 나머지 알파벳 문자는 자음으로 간주됩니다.
    • 프로그램에서 각 문자가 모음인지 자음인지 구분하는 방법은 간단하게 if 조건문으로 확인할 수 있습니다.
  1. 대소문자 처리:
    • 문제에서 대소문자 구분이 없으므로, 입력된 문자를 모두 소문자로 변환하여 처리할 수 있습니다. 이는 tolower() 함수를 사용하여 쉽게 처리할 수 있습니다.
  1. 문자열 순회:
    • 주어진 문자열을 하나씩 순회하면서 알파벳 문자를 확인한 후, 모음과 자음을 각각 카운트합니다. 공백이나 숫자는 무시합니다.

입력:

  • input.txt 파일의 첫 번째 줄에는 문자열이 주어집니다.
입력 (input.txt)출력 (output.txt)

문제 17: 두 문자열의 공통 문자 찾기

문제 설명:

  • 두 개의 문자열이 주어지면, 이 문자열에서 공통으로 포함된 문자를 찾아서 출력하는 프로그램을 작성하세요.
  • 문자열의 길이는 첫번째 문자열과 두번째 문자열 각각 100글자 입니다.
  • 문자열은 input.txt 파일에서 읽어오며, 출력은 output.txt 파일에 저장해야 합니다.

팁:

  • 두 번째 문자열에서 문자를 한 번만 사용해야 하는 이유:
    • 두 문자열에 같은 문자가 여러 번 나올 수 있지만, 두 번째 문자열에서 그 문자를 한 번 사용했으면 다시 사용하면 안 됩니다. 이를 처리하지 않으면 중복된 문자들이 잘못 출력될 수 있습니다.
    • 예를 들어, 첫 번째 문자열에 's'가 2번 있고, 두 번째 문자열에 's'가 1번 있다면, 공통 문자는 한 번만 출력되어야 합니다.
    • 예를 들어, 첫 번째 문자열에 's'가 2번 있고, 두 번째 문자열에 's'가 2번 있다면, s는 두 번 출력 됩니다.
  • 배열을 사용한 중복 문자 처리 방법:
    • 두 번째 문자열의 각 문자가 이미 사용되었는지를 기록하기 위해, 배열을 사용하여 확인할 수 있습니다.
    • 두 번째 문자열에서 문자가 발견되면, 그 인덱스에 해당하는 배열 값을 true로 설정합니다. 다음에 같은 문자가 발견되면, 이 배열을 확인하여 이미 사용된 문자임을 알 수 있습니다.
  • 구체적인 처리 과정:
    • 두 번째 문자열의 문자를 사용하면, 그 문자가 어느 위치에서 사용되었는지 기록합니다. 예를 들어, 배열의 각 인덱스는 두 번째 문자열의 문자가 나타난 위치를 의미합니다.
    • 첫 번째 문자열의 각 문자를 순차적으로 비교하면서, 두 번째 문자열에서 아직 사용되지 않은 동일한 문자가 있는지 확인합니다. 동일한 문자가 있다면 해당 문자는 사용되었다고 표시하고, 결과에 포함시킵니다.

입력:

  • input.txt 파일의 첫 번째 줄에는 첫 번째 문자열이 주어집니다.
  • 두 번째 줄에는 두 번째 문자열이 주어집니다.
