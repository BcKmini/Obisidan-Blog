## Normal_exit.c
```c
# include <stdlib.h>
int main()
{
exit(0);
}
```

## Error_exit.c
```c
#include <stdlib.h>
#include <stdio.h>
int main()
{
fprintf(stderr, "Error.\n");
exit(1);
}
```

## Fatal_error_exit.c
```c
#include <stdio.h>
int main()
{
int *i, k;

// *i = 3;
// j[10000] = 9;
// k = 7 / 0;

}
```
